# Obrázky / Images

Tato složka je určena pro fotografie lampičky **Vltavínový strážce**.

## Očekávané soubory

| Název souboru | Popis | Doporučené rozlišení |
|--------------|-------|----------------------|
| `hero.jpg` | Hlavní foto (hero sekce) | min. 1920 × 1080 px |
| `story.jpg` | Foto procesu / příběhu | min. 800 × 600 px |
| `gallery-1.jpg` | Galerie – 1. foto (velký formát) | min. 1200 × 900 px |
| `gallery-2.jpg` | Galerie – 2. foto | min. 800 × 600 px |
| `gallery-3.jpg` | Galerie – 3. foto | min. 800 × 600 px |
| `gallery-4.jpg` | Galerie – 4. foto | min. 800 × 600 px |
| `gallery-5.jpg` | Galerie – 5. foto | min. 800 × 600 px |

## Tipy

- Preferujte formát **JPEG** pro fotografie, **WebP** pro lepší kompresi.
- Maximální velikost jednoho souboru: **2 MB** (ideálně pod 500 KB).
- Pojmenujte soubory **bez diakritiky a mezer**, používejte pomlčky: `lampicka-detail.jpg`.
- Pro další ořez a optimalizaci použijte [Squoosh](https://squoosh.app) nebo [TinyPNG](https://tinypng.com).

## Po nahrání obrázků

Aktualizujte `src` atributy v `index.html` na odpovídající cesty.  
Každé místo je označeno komentářem:

```html
<!-- Vyměňte src za cestu k vaší hero fotografii -->
<img src="images/hero.jpg" alt="Vltavínový strážce – pohled zblízka" />
```
